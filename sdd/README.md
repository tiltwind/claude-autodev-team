# SDD

Spec-Driven Development 技能集

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/tiltwind/claude-autodev-team/main/sdd/scripts/install.sh | bash
```

## 技能说明

### sdd-lite

轻量 AI Agent Harness / checkpoint-driven coding 技能。适用于高输入、高频、多轮的代码与 agentic coding 任务。

## 使用方式

安装后，在 Claude Code 中通过斜杠命令调用：

```
/sdd-lite 修复用户登录后 session 过期的问题
/sdd-lite 重构订单模块的状态机
```
